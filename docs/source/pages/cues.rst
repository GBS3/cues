Meet the Cues
=============

This page contains a list of the cues that are currently available. If you would like to see a visual representation of each cue, you can see more in the gallery.

If you would like to see an interactive example of each type, you can run an example of each from your terminal by entering the following command (replacing ``<cue>`` with the type of cue)::

    python -m cues.<cue>

For example::

    python -m cues.survey
